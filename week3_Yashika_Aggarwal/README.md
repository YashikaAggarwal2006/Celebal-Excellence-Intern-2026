## 📝 Assignment Project

### Country Intelligence System

Developed an end-to-end Country Intelligence System using machine learning techniques to analyze socio-economic and health indicators of countries.

#### Techniques Applied

* K-Means Clustering
* DBSCAN
* Principal Component Analysis (PCA)
* Random Forest
* XGBoost
* Feature Importance Analysis

#### Objectives

* Segment countries into meaningful groups based on development indicators.
* Evaluate clustering quality using Silhouette Score.
* Visualize country segments using PCA.
* Predict cluster membership using classification models.
* Generate actionable socio-economic insights from the data.
