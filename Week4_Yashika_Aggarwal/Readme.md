# CIFAR-10 Image Classification using ANN and CNN

## Overview

This project implements and compares Artificial Neural Networks (ANN) and Convolutional Neural Networks (CNN) on the CIFAR-10 image classification dataset.

## Dataset

* CIFAR-10
* 50,000 Training Images
* 10,000 Test Images
* 10 Classes

## Technologies Used

* Python
* TensorFlow / Keras
* NumPy
* Pandas
* Matplotlib

## Models Implemented

### ANN

* Dense (512)
* Dropout (0.3)
* Dense (256)
* Dense (10, Softmax)

### CNN

* Conv2D (32)
* Conv2D (64)
* Conv2D (128)
* MaxPooling Layers
* Dense (128)
* Dropout (0.3)
* Dense (10, Softmax)

## Results

* Trained for 10 epochs
* Compared ANN and CNN validation accuracy
* Generated accuracy comparison plots
* Evaluated final test accuracy of both models

* Validation Curve (ANN vs CNN)
* Test Accuracy Comparison DataFrame
* Performance Analysis
